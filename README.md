# OpenClaw for fnOS (飞牛 NAS)

OpenClaw 个人 AI 助手的飞牛 NAS 安装包

## 安装说明

1. 在飞牛应用商店中上传此 FPK 包
2. 点击安装
3. 配置数据目录（默认 /Data/openclaw）
4. 访问 http://<飞牛 IP>:18789 完成初始化

## 功能特性

- 🤖 个人 AI 助手
- 💬 支持多消息渠道（飞书/微信/Telegram 等）
- 🧠 支持多种 AI 模型（OpenAI/DeepSeek/通义千问等）
- 📦 Docker 容器化部署
- 🔒 数据本地存储

## 技术栈

- Node.js 22+
- Docker
- OpenClaw Gateway

## 资源占用

- CPU: 轻量（闲置时几乎不占用）
- 内存：约 200-500MB
- 存储：约 1GB（含工作区）
